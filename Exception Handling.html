1. Java Program that Throws and Catches an Exception

2. Java Program that Throws an Exception if the Number is Odd

public class OddNumberException {
    public static void main(String[] args) {
        try {
            checkEven(3); // Change this number to test
        } catch (IllegalArgumentException e) {
            System.out.println("Caught Exception: " + e.getMessage());
        }
    }

    static void checkEven(int number) {
        if (number % 2 != 0) {
            throw new IllegalArgumentException("The number is odd.");
        }
    }
}

This program includes a method checkEven that throws an IllegalArgumentException if the provided number is odd. The main method handles the exception.

3. Java Program to Read a File and Throw an Exception if Not Found

import java.io.File;
import java.io.FileNotFoundException;
import java.util.Scanner;

public class FileReadException {
    public static void main(String[] args) {
        try {
            readFile("nonexistent.txt");
        } catch (FileNotFoundException e) {
            System.out.println("Caught Exception: " + e.getMessage());
        }
    }

    static void readFile(String fileName) throws FileNotFoundException {
        File file = new File(fileName);
        Scanner scanner = new Scanner(file);
        // Reading file contents
        while (scanner.hasNextLine()) {
            System.out.println(scanner.nextLine());
        }
        scanner.close();
    }
}

This program tries to read a file named nonexistent.txt. If the file does not exist, a FileNotFoundException is thrown, which is then caught in the main method.

4. Java Program to Read a List of Numbers from a File and Throw an Exception if Any are Positive

import java.io.File;
import java.io.FileNotFoundException;
import java.util.Scanner;

public class PositiveNumberException {
    public static void main(String[] args) {
        try {
            checkPositiveNumbers("numbers.txt");
        } catch (Exception e) {
            System.out.println("Caught Exception: " + e.getMessage());
        }
    }

    static void checkPositiveNumbers(String fileName) throws Exception {
        File file = new File(fileName);
        Scanner scanner = new Scanner(file);
        while (scanner.hasNextInt()) {
            int number = scanner.nextInt();
            if (number > 0) {
                throw new Exception("Found a positive number: " + number);
            }
        }
        scanner.close();
    }
}

This program reads numbers from a file named numbers.txt. It throws an exception if any of the numbers are positive.

6. Java Program to Read a List of Integers and Throw an Exception if Any are Duplicates

import java.util.HashSet;
import java.util.Scanner;

public class DuplicateNumberException {
    public static void main(String[] args) {
        try {
            checkDuplicates();
        } catch (Exception e) {
            System.out.println("Caught Exception: " + e.getMessage());
        }
    }

    static void checkDuplicates() throws Exception {
        Scanner scanner = new Scanner(System.in);
        HashSet numbers = new HashSet<>();
        System.out.println("Enter integers (enter -1 to stop):");
        while (true) {
            int number = scanner.nextInt();
            if (number == -1) break;
            if (!numbers.add(number)) {
                throw new Exception("Duplicate found: " + number);
            }
        }
        scanner.close();
    }
}

This program checks for duplicate integers input by the user. It throws an exception if a duplicate is found.

7. Java Program to Throw an Exception if String Does Not Contain Vowels

public class VowelException {
    public static void main(String[] args) {
        try {
            checkVowels("xyz");
        } catch (Exception e) {
            System.out.println("Caught Exception: " + e.getMessage());
        }
    }

    static void checkVowels(String str) throws Exception {
        if (!str.matches(".*[aeiouAEIOU].*")) {
            throw new Exception("The string does not contain vowels.");
        }
    }
}

This program checks if a given string contains vowels. If not, it throws an exception.

8. Java Program to Connect to Database and Throw Exception if Not Successful

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class DatabaseConnectionException {
    public static void main(String[] args) {
        try {
            connectToDatabase();
        } catch (SQLException e) {
            System.out.println("Caught Exception: " + e.getMessage());
        }
    }

    static void connectToDatabase() throws SQLException {
        String url = "jdbc:mysql://localhost:3306/mydb";
        String user = "username";
        String password = "password";
        Connection connection = DriverManager.getConnection(url, user, password);
        System.out.println("Connected to database successfully.");
        connection.close();
    }
}

This program attempts to connect to a database. If the connection fails, it throws a SQLException.

10. Java Program to Throw SQL Queries with Exceptions

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;
import java.sql.Statement;

public class SQLQueryException {
    public static void main(String[] args) {
        try {
            executeSQL("INSERT INTO my_table (column1) VALUES ('value1')");
        } catch (SQLException e) {
            System.out.println("Caught Exception: " + e.getMessage());
        }
    }

    static void executeSQL(String sql) throws SQLException {
        Connection connection = DriverManager.getConnection("jdbc:mysql://localhost:3306/mydb", "username", "password");
        Statement statement = connection.createStatement();
        statement.executeUpdate(sql);
        System.out.println("SQL query executed successfully.");
        statement.close();
        connection.close();
    }
}

This program executes an SQL query and throws an exception if it fails.

12. Java Program to Show an Example of Multiple Catch

public class MultipleCatchExample {
    public static void main(String[] args) {
        try {
            int[] arr = new int[2];
            arr[5] = 10; // This will cause ArrayIndexOutOfBoundsException
        } catch (ArrayIndexOutOfBoundsException e) {
            System.out.println("Caught ArrayIndexOutOfBoundsException: " + e.getMessage());
        } catch (Exception e) {
            System.out.println("Caught Exception: " + e.getMessage());
        }
    }
}

This program shows how to handle multiple exceptions using multiple catch blocks. Here, it specifically catches ArrayIndexOutOfBoundsException.

13. Java Program to Show an Example of Nested Try Block

public class NestedTryExample {
    public static void main(String[] args) {
        try {
            try {
                throw new Exception("Inner exception");
            } catch (Exception e) {
                System.out.println("Caught Inner Exception: " + e.getMessage());
            }
        } catch (Exception e) {
            System.out.println("Caught Outer Exception: " + e.getMessage());
        }
    }
}

This program demonstrates a nested try block where the inner block can throw an exception that is caught in the inner catch.
